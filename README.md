### Структура "Матрьошки" (Композиція)
Текст у програмі не є просто рядком, він складається з:
1.  **Text** — масив об'єктів `Sentence`.
2.  **Sentence** — масив об'єктів `SentenceElement` (інтерфейс).
3.  **Word** (реалізує `SentenceElement`) — масив об'єктів `Letter`.
4.  **Punctuation** (реалізує `SentenceElement`) — окремий символ розділового знаку.
5.  **Letter** — найменша одиниця (символ).

## Предметна область: Спортивний інвентар
Згідно з варіантом №6, реалізовано клас `SportEquipment`, який має 5 полів:
- `Text name` — назва (використовує об'єктну модель тексту).
- `String brand` — виробник.
- `double price` — ціна.
- `double weight` — вага.
- `int quality` — показник якості.
