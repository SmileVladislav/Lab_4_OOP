## Предметна область: Спортивний інвентар
Згідно з варіантом №6, реалізовано клас `SportEquipment`, який має 5 полів:
- `Text name` — назва (використовує об'єктну модель тексту).
- `String brand` — виробник.
- `double price` — ціна.
- `double weight` — вага.
- `int quality` — показник якості.
